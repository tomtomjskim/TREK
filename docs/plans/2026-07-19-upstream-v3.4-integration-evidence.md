## Residual warnings and decisions

- 공식 v3.4 fresh schema는 migration 151/152에서 이미 존재하는 `links` 2개와 `role`
  컬럼을 다시 추가하려다 non-fatal warning을 낸다. 공식 migration 파일은 변경하지 않고
  허용 warning을 테스트로 고정했다. 일반화 가능한 upstream fix 후보이다.
- server/client lint는 오류는 없지만 기존 warning backlog가 각각 1,946개/1,273개다.
  이번 통합에서 생긴 shared Prettier 오류 46개는 제거했다.
- Vite build는 main chunk와 HEIC chunk 크기, ineffective dynamic import를 경고한다.
  기능 blocker는 아니지만 성능 개선 후보이다.
- v3.4가 새로 추가한 Catalan에는 포크의 Google/map locale 키를 English fallback으로
  채웠다. key parity는 맞지만 native Catalan 번역 검토가 남아 있다.
- Fold collision은 폭/패널 조합 unit regression과 planner browser mount로 검증했다.
  실제 Fold 7 장치의 v3.4 authenticated planner 최종 확인은 배포 후 사용자 인수로
  남긴다.
